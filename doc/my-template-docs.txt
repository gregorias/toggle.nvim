Toggle is a modern, extensible Neovim plugin for toggling options ala
unimpaired.vim <https://github.com/tpope/vim-unimpaired> written in Lua.


REQUIREMENTS                                   *my-template-docs-requirements*

- Neovim 0.10+
- Optional plugin dependencies:
    - Which Key <https://github.com/folke/which-key.nvim>


INSTALLATION                                   *my-template-docs-installation*

Install the plugin with your preferred package manager, such as [Lazy]:

>lua
    {
      "gregorias/toggle.nvim",
      config = true,
    }
<
